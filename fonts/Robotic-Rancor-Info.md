# Robotic Rancor
Robotic Rancor is a 7pt pixel art font designed with futuristic looking lettering and multiple font weight variations.

## Author
[VEXED](https://v3x3d.itch.io/)

## Basic Info
- Base Glyph Height: 7px
- Scales: 7, 14, 21, 28, etc
- Ascender Max: 3px
- Descender Max: 2px
